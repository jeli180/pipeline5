FPGA BREAKOUT PIN REFERENCE FOR top.sv
======================================

Signal path:
physical FPGA package pin -> ice40hx8k wrapper port -> top.sv port

Notes:
- hz100 is generated inside ice40hx8k from hwclk.
- reset is generated inside ice40hx8k using reset_on_start.
- txdata/rxdata/txclk/rxclk/txready/rxready connect between top.sv and the UART logic inside ice40hx8k.
- Rx and Tx are the external physical UART pins.
- CTSn and DCDn are external UART control pins driven by ice40hx8k.
- The Makefile synthesizes ice40hx8k as the board-level top, so the physical pin names come from the pinmap.pcf constraints.

CLOCK / RESET
-------------
top.sv signal: hz100
Physical source: hwclk on FPGA pin J3
Notes: hz100 is not directly pinned. It is generated inside ice40hx8k by dividing hwclk.

top.sv signal: reset
Physical source: reset_on_start logic inside ice40hx8k
Manual reset condition: pb[3] && pb[0] && pb[16]
Physical pins involved:
  pb[3]  -> C5
  pb[0]  -> C3
  pb[16] -> B9

UART-RELATED top.sv SIGNALS
---------------------------
top.sv signal: txdata[7:0]
Physical FPGA pin: none directly
Notes: Internal bus from top.sv to the UART module inside ice40hx8k.

top.sv signal: rxdata[7:0]
Physical FPGA pin: none directly
Notes: Internal bus from the UART module inside ice40hx8k to top.sv.

top.sv signal: txclk
Physical FPGA pin: none directly
Notes: Internal handshake/control signal from top.sv to ice40hx8k UART logic.

top.sv signal: rxclk
Physical FPGA pin: none directly
Notes: Internal handshake/control signal from top.sv to ice40hx8k UART logic.

top.sv signal: txready
Physical FPGA pin: none directly
Notes: Internal ready signal from ice40hx8k UART logic to top.sv.

top.sv signal: rxready
Physical FPGA pin: none directly
Notes: Internal ready signal from ice40hx8k UART logic to top.sv.

External UART physical pins used by ice40hx8k:
  Rx -> B10
  Tx -> B12

External UART control pins used by ice40hx8k:
  DCDn -> A15
  CTSn -> B14


BOARD-LEVEL WRAPPER SIGNALS NOT DIRECTLY IN top.sv
--------------------------------------------------
ice40hx8k signal: hwclk
Physical FPGA pin: J3
Notes: Source clock input. Used to generate top.sv hz100.

ice40hx8k signal: Rx
Physical FPGA pin: B10
Notes: External UART receive input.

ice40hx8k signal: Tx
Physical FPGA pin: B12
Notes: External UART transmit output.

ice40hx8k signal: DCDn
Physical FPGA pin: A15
Notes: UART control output.

ice40hx8k signal: CTSn
Physical FPGA pin: B14
Notes: UART control output.
